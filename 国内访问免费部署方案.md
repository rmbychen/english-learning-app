# 国内访问免费部署方案

由于 Vercel 在中国大陆访问可能较慢或被限制，这里提供几个**完全免费**的替代方案。

## 🎯 推荐方案（按优先级排序）

### 方案一：Cloudflare Pages（最推荐 ⭐⭐⭐⭐⭐）

**优势：**
- ✅ 完全免费
- ✅ 全球 CDN，在中国访问速度较好
- ✅ 自动部署（连接 GitHub）
- ✅ 支持自定义域名
- ✅ 无需信用卡

**部署步骤：**

1. **注册 Cloudflare 账号**
   - 访问：https://dash.cloudflare.com/sign-up
   - 使用邮箱注册（免费）

2. **连接 GitHub**
   - 登录后，进入 **Workers & Pages**
   - 点击 **Create application** → **Pages** → **Connect to Git**
   - 选择 GitHub，授权后选择仓库 `rmbychen/english-learning-app`

3. **配置构建设置**
   - **Project name**: `english-learning-app`
   - **Production branch**: `main`
   - **Build command**: `cd english-learning-app && npm install && npm run build`
   - **Build output directory**: `english-learning-app/dist`
   - **Root directory**: 留空（或设置为 `/`）

4. **部署**
   - 点击 **Save and Deploy**
   - 等待部署完成（通常 2-3 分钟）

5. **获取访问地址**
   - 部署完成后会得到一个 URL，例如：
     - `https://english-learning-app.pages.dev`
   - 这个地址在中国访问速度会好很多！

---

### 方案二：GitHub Pages（简单但功能有限）

**优势：**
- ✅ 完全免费
- ✅ 简单易用
- ✅ 自动部署

**限制：**
- ⚠️ 只支持静态网站
- ⚠️ 构建需要在本地或 GitHub Actions 完成
- ⚠️ 访问速度可能一般

**部署步骤：**

1. **使用 GitHub Actions 自动构建**
   - 在项目根目录创建 `.github/workflows/deploy.yml`
   - 我已经为你准备好了配置文件（见下方）

2. **启用 GitHub Pages**
   - 在 GitHub 仓库设置中
   - **Settings** → **Pages**
   - **Source**: 选择 `GitHub Actions`
   - 保存后会自动部署

3. **获取访问地址**
   - `https://rmbychen.github.io/english-learning-app/`

---

### 方案三：Netlify（备选）

**优势：**
- ✅ 完全免费
- ✅ 自动部署
- ✅ 在中国访问可能比 Vercel 稍好

**注意：**
- ⚠️ 在中国访问可能仍然较慢
- ⚠️ 但比 Vercel 好一些

**部署步骤：**

1. 访问：https://app.netlify.com
2. 使用 GitHub 登录
3. 导入项目，选择仓库
4. 配置：
   - **Base directory**: `english-learning-app`
   - **Build command**: `npm run build`
   - **Publish directory**: `english-learning-app/dist`
5. 部署

---

### 方案四：腾讯云静态网站托管（国内，需实名）

**优势：**
- ✅ 在中国访问速度最快
- ✅ 有免费额度（每月 10GB 流量）
- ✅ 支持自定义域名

**限制：**
- ⚠️ 需要实名认证
- ⚠️ 需要微信/QQ 登录

**部署步骤：**

1. 访问：https://cloud.tencent.com
2. 注册并实名认证
3. 进入 **对象存储 COS** → **静态网站托管**
4. 创建存储桶，上传 `dist` 目录内容
5. 配置静态网站托管
6. 获取访问地址
