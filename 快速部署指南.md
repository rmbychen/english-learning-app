# 🚀 快速部署指南

## 第一步：推送到 GitHub

### 1. 在 GitHub 创建仓库

访问 https://github.com/new 创建新仓库：
- 仓库名称：例如 `english-learning-app`
- 选择 Public 或 Private
- **不要**勾选任何初始化选项
- 点击 "Create repository"

### 2. 执行推送命令

在项目目录执行以下命令（替换 YOUR_USERNAME 和 YOUR_REPO_NAME）：

```bash
cd /Users/chenlei/2025/AI/package_3

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 节奏英语学习应用"

# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 重命名分支为 main
git branch -M main

# 推送到 GitHub
git push -u origin main
```

## 第二步：部署到 Vercel（推荐，2分钟完成）

### 方法 1: 通过网页（最简单）

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击 "Sign Up" 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "Add New Project"
   - 选择你刚推送的 GitHub 仓库
   - 点击 "Import"

3. **配置项目**
   ```
   Framework Preset: Vite
   Root Directory: english-learning-app
   Build Command: npm run build
   Output Directory: dist
   ```

4. **部署**
   - 点击 "Deploy"
   - 等待 1-2 分钟

5. **获取访问地址**
   - 部署完成后会显示：`https://your-project.vercel.app`
   - 这个地址可以立即访问！

### 方法 2: 使用命令行

```bash
# 安装 Vercel CLI
npm i -g vercel

# 进入项目目录
cd english-learning-app

# 部署
vercel

# 生产环境部署
vercel --prod
```

## 第三步：配置 Supabase（重要）

部署后需要在 Supabase 中配置：

1. **登录 Supabase Dashboard**
   - 访问 https://supabase.com/dashboard

2. **配置重定向 URL**
   - 进入 **Authentication** -> **URL Configuration**
   - 在 **Redirect URLs** 中添加你的部署地址：
     ```
     https://your-project.vercel.app
     https://your-project.vercel.app/**
     ```
   - 点击 "Save"

## ✅ 完成！

现在你的应用已经可以访问了：
- **GitHub 仓库**: `https://github.com/YOUR_USERNAME/YOUR_REPO_NAME`
- **访问地址**: `https://your-project.vercel.app`

## 📝 后续更新

每次更新代码后，只需：

```bash
git add .
git commit -m "Update: 描述更改"
git push
```

Vercel 会自动重新部署！
