# 艾宾浩斯遗忘曲线和间隔重复算法(SRS)研究计划

## 研究目标
深入研究艾宾浩斯遗忘曲线和间隔重复算法的科学原理和实际应用，为学习算法和记忆增强技术提供理论基础和实践指导。

## 研究范围
1. 艾宾浩斯遗忘曲线的数学模型和实际应用
2. 主流SRS算法（如SM-2、SM-17、FSRS等）的原理和实现
3. 如何根据用户表现调整复习间隔
4. 短期记忆向长期记忆转化的机制
5. 个性化复习时间预测算法
